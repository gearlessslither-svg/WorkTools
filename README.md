# WorkTools

Reusable work tools, skills, and process resources.

## Categories

- `ppt-report-generation/`: PPT report generation tools and Codex skill for learning a previous source-folder plus finished-PPT pair, then generating a new PPT from a new source folder.
- `mullvad-speed-guard/`: Local Mullvad VPN health guard, relay inventory, LaunchAgent installers, and localhost control panel for unattended VPN recovery.

Generated reports, PDFs, PPTX outputs, source documents, images, audio, and video files are intentionally excluded from this repository.
